### ✅ Cowork is installed — but something stopped working?

Use the After Set Up sections. Each tier covers a specific category of error.

| Tier | What it covers |
|---|---|
| **Tier 1 — Networking & Plugins** | API unreachable, session file corruption, plugin marketplace failures |
| **Tier 2 — Scheduled Tasks** | Task creation fails, tasks that don't run, storage path conflicts |
| **Tier 3 — Connectors & Add-ins** | MCP registry errors, OAuth failures, Office add-in bugs |
| **Tier 4 — Known Issues** | Confirmed bugs or OS limits that require Anthropic to fix |
